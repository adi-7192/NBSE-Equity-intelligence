## Data Flow

**Market Data Update Flow:**

1. **Client** calls `POST /api/update-data` via `UpdateButton` component
   - Location: `components/update-button.tsx` line 41

2. **Route Handler** (`app/api/update-data/route.ts`):
   - Archives current live data
   - Calls `generateText()` from Vercel AI SDK
   - Sends structured prompt to Claude Opus 4.6
   - Receives structured JSON response (Zod-validated)

3. **Claude Generates:**
   - Market baseline (Nifty 50, indices, commodities, FX, yields)
   - Alert cards (3 market-moving events)
   - Macro pulse (5 key indicators)
   - Sector rankings (8 sectors with ratings)
   - Watch list (3 upcoming events)

4. **Data Persistence:**
   - New data saved to `currentLiveData` in `lib/archive.ts`
   - Previous state saved to `archiveHistory`
   - Returns success response with summary

5. **Display:**
   - `app/page.tsx` loads live or archived data
   - Renders via `DashboardClient` component
   - Archives page displays historical snapshots

## Data Schema

**Market Data Structure** (Generated by Claude, validated with Zod):
- `marketBaseline`: Index values, FX rates, commodity prices, yields
- `alertCards`: Array of market events with investment implications
- `macroPulse`: Key macro indicators (rates, flows, inflation, collections, capex)
- `sectorRankings`: 8-sector investment ratings (Defence, Pharma, Banking, Infrastructure, Energy, Auto, FMCG, IT)
- `stockPicks`: Manually curated equity selections
- `picksUnderReview`: Candidates for future inclusion
- `watchNextWeek`: Upcoming catalysts and events
